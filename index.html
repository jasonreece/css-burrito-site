css-burrito

a customizable Sass template and generator

npm install -g css-burrito

Starter Template

css-burrito is a SMACSS based starter template for your Sass projects.

Add css-burrito to your project by running: 

burrito -n [folder-name] [file-name] [path-to-sass-directory]

If none of the optional arguments are passed in, the folder name will be stylesheets, the file name will be application.scss, and the path will be the current directory.

The default template structure looks like this: 

stylesheets/
  global/
    _settings.scss
    _utilities.scss
    _base.scss
    _layout.scss
    _skin.scss
    _typography.scss

  libs/
    _library-variable-overrides.scss
    _normalize.scss

  modules/
    _modules.scss

  application.scss

Generate

Tired of manually adding module files into your project?

Add new module files to your project by running: 

burrito -m (fileName[s])

Remove module files to your project by running: 

burrito -r (fileName[s])

List your module files by running: 

burrito -l

Customize

Want to use the generator to add module files, without being stuck with our template? That's cool, you'll just need to set up your config.

Just add your file structure to a css-burrito-config file, and run: 

burrito -g

Now you can generate new module files to your project by running: 

burrito -m (fileName[s])
